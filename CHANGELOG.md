# Changelog

## Unreleased

- Added Explorer-style bookmark manager with folder-like category/tag navigation and inline editing.

## 0.1.0 - First Public Release

- Self-hosted FastAPI web app.
- SQLite database and local media storage.
- Public and private bookmark visibility.
- Categories and tags.
- Browser extension for quick saving.
- Website previews and screenshots.
- YouTube embeds for bookmark-only links.
- Internal `yt-dlp` media downloads.
- Optional ReClip-compatible downloader fallback.
- JSON import/export.
- Admin, label management, and app stats pages.
